## ‚ú® Key Features
* **User Authentication:** Secure login/signup using **Firebase Authentication**.
* **JWT Security:** Implemented **JSON Web Tokens (JWT)** for secure API communication.
* **CRUD Operations:** Users can Create, Read, Update, and Delete job posts and applications.
* **Axios Interceptors:** Custom **Axios Secure Instance** to handle 401/403 errors and auto-logout.
* **Dynamic Search & Filter:** Easily filter jobs by salary ranges and job titles.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using **Tailwind CSS**.

## üõ† Tech Stack
* **Frontend:** HTML5, Tailwind CSS, React.js
* **Routing:** React Router DOM (v6)
* **Authentication:** Firebase Auth
* **Backend Communication:** Axios (with Interceptors)
* **Security:** JWT (JSON Web Token)
* **State Management:** React Hooks (useState, useEffect, useContext)

## üèó Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/job-portal.git](https://github.com/your-username/job-portal.git)
    ```
2.  **Install dependencies:**
    ```bash

    ```

    ```bash
    npm run dev
    ```
